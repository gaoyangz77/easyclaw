为什么选 DlxAI？

市面上没有第二个产品这样定位。

不像 Jan / Msty

它会主动来找你。在你常用的聊天 App 里收发消息，而不是你打开 app 才能跟它说话。

不像 Botpress / Voiceflow

它是你的，不是你客户的。个人 AI，跟随你的身份和偏好，没有 visual flow editor。

不像 n8n / Make

你不用画流程图。规则用一句话说清楚，自动编译成策略和技能，立即生效。

三步开聊

1

下载安装

Mac / Windows / Linux 一键安装包，启动后驻留系统托盘。

接入聊天 App
